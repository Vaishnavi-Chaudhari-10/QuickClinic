# 🏥 QuickClinic – Skip the Wait

QuickClinic is a simple, responsive landing page for a healthcare booking platform.  
It allows users to view live wait times, doctor availability, and book appointments instantly.

## 📌 Features
- ✅ Responsive design (mobile & desktop friendly)
- ✅ Real-time wait time & doctor availability sections
- ✅ Instant booking CTA buttons
- ✅ Contact information
- ✅ Clean HTML, CSS, and JS structure

---

## 🛠 Tech Stack
- **HTML5** – Structure
- **CSS** – Styling & responsiveness
- **JavaScript (ES6)** – Interactivity
